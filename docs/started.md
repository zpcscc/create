## 使用

#### 全局安装

```shell
npm install -g @dxsixpc/create
```

全局安装完成后，可直接使用 create 命令

```shell
create
```

![create](../static/img/create.png)

#### 通过 npx 直接使用

```shell
npx @dxsixpc/create
```

![npx_@dxsixpc-create](../static/img/npx_@dxsixpc-create.png)

#### 命令参数

```shell
npx @dxsixpc/create --help
```
